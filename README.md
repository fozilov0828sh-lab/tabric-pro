# Tabric Pro

Professional Congratulations and Relationships Platform

## Features
- User & Admin panels
- Professional design with Glassmorphism
- Dark/Light mode support
- Real-time notifications
- Order management system
- Responsive design

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Zustand

## Getting Started

```bash
npm install
npm run dev
```

Server runs on http://localhost:3001

## Build

```bash
npm run build
npm run preview
```
